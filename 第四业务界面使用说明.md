# 第四业务界面使用说明

## 概述
第四业务界面用于生成评估数据表，分为两个子页面：
1. **常规油评估数据表**：处理常规油井的评估数据
2. **页岩油评估数据表**：处理页岩油井的评估数据

## 功能流程

### 常规油评估数据表处理流程

**第一步：调整历史数据**
- 在表f（上年度油-常规评估数据表）的基础上
- 根据表b（年度油井SEC单元变化表）中油井的SEC单元归属变化
- 将表g（油井历史生产数据）中的历史数据，在表f中的SEC单元汇总中进行增减调整
- 形成更新后的评估数据表（数据时间仍截止至上一年评估月）

**第二步：加入新增PD评估井**
- 将表c（扩边/PUD转PDP/PDNP数据表）和表j（老区新井数据表）的井
- 结合表h（A2数据表，连续12个月份数据）中的连续月份数据
- 根据"SEC油田"、"SEC单元"属性，汇总入上一步的评估数据表
- 评估单元字段为SEC单元

**第三步：更新老PD评估井年度数据**
- 根据表a（上年度SEC数据表）中的井号
- 在表h（A2数据表）中检索本年度连续12个月生产数据（如202410~202509）
- 按照表e（本年度SEC数据表）中油井的"SEC油田"、"SEC单元"单元关系汇总
- 将汇总数据与上一步的评估数据表融合（按年月数列顺序，"SEC油田"、"SEC单元"合并）
- 生成新的本年度评估数据表

### 页岩油评估数据表处理流程

**第四步：页岩油评估数据表更新**
- 在表f（上年度油-页岩油评估数据表）的基础上（评估单元字段为单井井号）
- 将表c（扩边/PUD转PDP/PDNP数据表）中页岩油属性的井号汇总
- 在表h（A2数据表）中查询检索本年度连续12个月生产数据
- 按年月顺序与表f融合，形成新的本年度页岩油评估数据表

## 注意事项

1. **油井分类处理**：单元调整变化、数据延续合并等操作仅限于"常规"油井，对"页岩油"类油井不进行此类操作

2. **数据规模考虑**：
   - 单月井数数据条约100,000条
   - 12个月份总计约1,200,000条
   - 系统已优化本地存储读取效率

3. **数据表说明**：
   - 表a：上年度SEC数据表（如202409）
   - 表b："****年度油井SEC单元变化表"（如2025年度）
   - 表c：上年度"扩边/PUD转PDP/PDNP"数据表（如2024年）
   - 表e：本年度SEC数据表（如202509，第三业务界面生成结果）
   - 表f：上年度油-常规/页岩油评估数据表
   - 表g：表b中油井历史生产数据（开井至上年度评估月）
   - 表h：上年度对应月份至本年度对应月份的A2数据表（12个月数据）
   - 表j：本年度"老区新井数据表"

## 使用方法

1. 选择评估类型（常规油或页岩油）
2. 上传相应的数据文件
3. 点击处理按钮
4. 查看处理结果并下载评估数据表

## 文件上传要求

- Excel格式（.xlsx或.xls）
- 包含必要的字段（井号、SEC油田、SEC单元等）
- 数据完整性校验

## 输出结果

- 生成完整的评估数据表
- 包含油气田、评估单元、生产时间、油产量、液产量、油井开井数等字段
- 自动保存并提供下载链接
